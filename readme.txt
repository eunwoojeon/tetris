javascript 런타임 : node.js
라이브러리
npm : 여러 라이브러리 다운로드 창구 (npm init -> enter -> entry point에서 사용할 js파일명 작성)
express : 웹 서버 구성 (npm install express)
nodemon : 서버 재부팅 (npm install -g nodemon) (-g는 global 해당 컴퓨터의 다른 폴더에서도 적용)
bootstrap : HTML/CSS 라이브러리

실행법 : node server.js or nodemon server.js
종료법 : ctrl + c
